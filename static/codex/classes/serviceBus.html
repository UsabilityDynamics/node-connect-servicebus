Defined in lib/connect-servicebus.js:29

Options Instance.

Mixing OS into parent's settings and set options with defaults. Extend original settings, new options, and defaults into the injected instance of Object Settings. Return settings object.

Object Settings overwrites the "get" and "set" functions and hooks into events triggering events such as "set.something" whenever this.set( 'something', 'blah' ) is used.
